# JavaScript Table Generator

A simple JavaScript project that generates a multiplication table dynamically.

## Features

* User enters a number using prompt
* Displays multiplication table from 1 to 10
* Uses HTML, CSS, and JavaScript
* Dynamic output using DOM manipulation

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Open the project folder
2. Run `index.html` in browser
3. Click the **Tables** button
4. Enter a number
5. View the multiplication table

## Project Structure

* `index.html`
* `style.css`
* `index.js`
